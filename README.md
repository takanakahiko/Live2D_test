# Live2D_test
Live2Dのチュートリアルを行っていくリポジトリです。


## SDKおよびモデルについて
DKおよびモデルの二次配布がグレーっぽいので、
リポジトリには含めていません。  
フォルダ構成を次のようにすれば動かせます。

## フォルダ構成

- Live2D_test（本リポジトリ）
  - simple.html  
  - src  
    - Simple.js  
- LIB
  - Live2D_SDK_WebGL_2.0.05_1_jp([ここ](http://sites.cybernoids.jp/cubism-sdk2/webgl2-1)からダウンロードして解凍)
